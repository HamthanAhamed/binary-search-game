📚 How to Play

🎯 Goal

Find the ONE switch that lights the bulb using as few checks as possible.

💡 Rules

  • Toggle switches ON/OFF by clicking them
  • Press CHECK to test your selection
  • Bulb lights up if the correct switch is ON
  • Win when exactly ONE switch is ON and it's correct
  • Each check costs 5 points · Hint costs 10 points

🔍 Binary Search Strategy

1. Turn ON half the switches → CHECK → if bulb ON, correct switch is in ON group

2. If bulb stays OFF, correct switch is in OFF group

3. Repeat with the smaller group until only one switch remains

📊 Optimal checks per level:
8 switches → 3 checks · 16 switches → 4 checks
32 switches → 5 checks · 64 switches → 6 checks
Binary Search halves the search space each step:
64 → 32 → 16 → 8 → 4 → 2 → 1 ✅
